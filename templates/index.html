How it works
Four layers of
intelligent protection

Raw sensor data transformed into actionable crop-saving decisions in under 2 seconds, 24 hours a day.

📡
IoT Sensor Network
Raspberry Pi / ESP32 nodes continuously measure soil moisture, temperature, humidity, and motion across every field zone via MQTT protocol.
👁️
YOLOv8 Object Detection
Camera feeds processed frame-by-frame using YOLOv8 to identify pests (aphids, caterpillars), wild animals (boars, deer, monkeys), and diseased crops in real time.
🔊
Automated Deterrents
Threats automatically activate non-lethal audio-visual deterrents while simultaneously sending SMS alerts and push notifications to the farmer's dashboard.
📊
Analytics Dashboard
Historical threat trends, pesticide-use reduction tracking, and environmental pattern analysis presented in a mobile-responsive farmer-friendly interface.
☁️
Edge + Cloud Architecture
Time-critical inference runs on-device for sub-second response. Long-term data syncs to cloud for seasonal pattern modeling and predictive alerts.
🌱
Sustainable by Design
Targeted detection reduces blanket pesticide use by 68%, cutting costs, protecting pollinators, and preserving soil health for future seasons.
